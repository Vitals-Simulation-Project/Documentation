Vitals Simulation

Introduction

Vitals Simulation is a simulation of a Search and Rescue operation using AI-powered drones. This project was created as part of UCF Computer Science Senior Design. The project is sponsored by Lockheed Martin.

Project Overview

The goal of VITALs (Visual Intelligent Tactical Assistant for Low-power operations) Simulation is to produce a proof-of-concept for conducting a Search and Rescue operation with fully autonomous drone swarm. The simulation is designed to demonstrate the capabilities of AI-powered drones and test various AI models and rescue strategies in a simulated environment. Users have the ability to view and override drone behavior throughout the simulation.
